#  Class 01 - 02 Primitive Types  

> Use strings to write the name of Javascript primitive types

**Use the following format as an example :**

~~~javascript
export const primitiveTypes =
    [
        // After this row
        "Type01",
        "Tyoe02",
        "Type03",
        "Type04",
        "Type05"

        // Before this row
    ]
~~~
